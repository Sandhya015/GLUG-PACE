WORKSHOP: Crafting the Web

Date: 11-07-2024 to 12-07-2024

The GLUG PACE organized an engaging workshop titled "Crafting the Web: A Hands-On Workshop on Web Development" on July 11th and 12th, 2024, specifically designed for second-year students in CSE, AIML, and EC. The workshop's goal was to offer a comprehensive insight into web development, covering frontend, backend, and database management systems like NoSQL (MongoDB), Node.js, Express, as well as the fundamentals of HTML and CSS. The agenda included hands-on sessions and initial configuration.

On the first day, participants delved into Express.js, an impressive library facilitating the creation of dynamic backend and full-stack web applications.

DAY 1: Participants learnt about the following topics:

  • Installing and configuring a fundamental web server using Express.
  • Setting up a route for the website.
  • Once a user accesses this route, our landing page or any HTML page is shown.
  • Establishing a basic structure for both the frontend and backend of the website.
  • Transferring data from the backend to the frontend.
  • Establishing the initial route and the landing page for the website.

DAY 2: Participants learnt about Database management system (No SQL) & frontend :

  • Setting up a MongoDB Atlas database in the cloud and adding data
  • Linking our backend to MongoDB Atlas
  • Retrieving data from the database using NodeJS backend
  • Sending data from the backend to the database
  • Transferring data from the backend to the frontend.
  • Designing a login page for the front end using HTML & CSS

The workshop was skillfully led by Mohammad Hisham Hassan, Mohammad Nihal, Mohammad Thameem Raz, Aakif Mohammad, and all the coordinators. Their invaluable guidance and expertise made complex concepts accessible and engaging.

During the closing Valedictory program, Professor Dr. Mohammed Hafeez M K and the staff coordinator of GLUG PACE honored the resource persons.

Participants praised the clear explanations and practical exercises that aided in understanding the basics of web development. The feedback was overwhelmingly positive, with many expressing interest in future workshops.

In conclusion the "Crafting the Web A hands - on workshop on Web development" workshop successfully provided participants with a foundational understanding of web development and its essential operations. The success of this workshop is a testament to the dedication and hard work of the organizing team and resource persons
